# Futuristic Web Developer Portfolio

A modern, futuristic portfolio website with glassmorphism design, neon accents, and smooth animations built with React, Tailwind CSS, and Framer Motion.

## 🚀 Features

- **Futuristic Design**: Glassmorphism effects, neon accents, and modern UI
- **Responsive Layout**: Works perfectly on all devices
- **Smooth Animations**: Powered by Framer Motion
- **Interactive Elements**: Hover effects, micro-interactions
- **Modern Tech Stack**: React 18, Vite, Tailwind CSS
- **Performance Optimized**: Fast loading and smooth interactions

## 🛠️ Tech Stack

- **React 18**: Modern component-based architecture
- **Vite**: Fast build tool and development server
- **Tailwind CSS**: Utility-first CSS framework with custom configuration
- **Framer Motion**: Production-ready animation library
- **React Icons**: Beautiful SVG icons
- **React Hook Form**: Form validation and management
- **React Type Animation**: Animated typing effects

## 🎨 Design Elements

- **Color Palette**: Dark background with neon blue (#00eeff), cyan (#00ffff), and purple (#bf00ff) accents
- **Typography**: Space Grotesk (headings), Inter (body text)
- **Glassmorphism**: Frosted glass effect cards and buttons
- **Animations**: Fade in, slide in, floating, and glow effects
- **Micro-interactions**: Hover states, button feedback

## 🏗️ Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd futuristic-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser to `http://localhost:3000`

## 🔧 Customization

### Updating Content
- Edit the components in `src/sections/` to update your personal information
- Replace placeholder images with your own
- Update project details in the Projects section

### Styling
- Modify colors in `tailwind.config.js`
- Adjust animations in `src/styles/globals.css`
- Update typography in the same file

### Adding New Sections
- Create a new component in `src/sections/`
- Import and add it to `src/App.jsx`
